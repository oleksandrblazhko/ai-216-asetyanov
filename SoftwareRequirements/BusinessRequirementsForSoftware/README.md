## Бізнес-вимоги до програмного продукту
- **Вік**: з 16 років, якщо користувач молодший за цей вік, то має бути нагляд з боку батьків, опікунів чи піклувальників.
- **Зайнятість**: будь-яка.
- **Стать**: неважливо.
- **Сімейний стан**: будь-який.
- **Фінансовий стан**: будь-який, за вийнятком, якщо користувач не вдається до послуг, що можуть бути платними.
- **Професійний статус**: будь-яка професія.
